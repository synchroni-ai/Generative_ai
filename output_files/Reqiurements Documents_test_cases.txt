**Test Cases:**

**TC_001**
*   **Title:** Valid Patient Registration
*   **Description:** Verify that a patient can successfully register with valid inputs.
*   **Steps:**
    1. Navigate to the registration page.
    2. Fill in all required fields with valid data.
    3. Submit the registration form.
*   **Expected Result:** The patient should be registered successfully, and a confirmation message should be displayed.

**TC_002**
*   **Title:** Invalid Patient Registration - Missing Fields
*   **Description:** Verify that the system handles missing required fields during patient registration.
*   **Steps:**
    1. Navigate to the registration page.
    2. Leave one or more required fields empty.
    3. Submit the registration form.
*   **Expected Result:** The system should display error messages for the missing fields and not allow registration.

**TC_003**
*   **Title:** Successful Patient Login
*   **Description:** Verify that a patient can log in with valid credentials.
*   **Steps:**
    1. Navigate to the login page.
    2. Enter valid username and password.
    3. Click the login button.
*   **Expected Result:** The patient should be logged in successfully, and the dashboard should be displayed.

**TC_004**
*   **Title:** Invalid Patient Login - Wrong Password
*   **Description:** Verify that the system handles incorrect passwords during patient login.
*   **Steps:**
    1. Navigate to the login page.

    3. Click the login button.
*   **Expected Result:** The system should display an error message indicating incorrect credentials.
