**Test Cases:**

*   **Test Case ID:** TC_001
    *   **Title:** Valid Patient Registration
    *   **Description:** Verify that a patient can successfully register with valid information.
    *   **Steps:**
        1. Navigate to the registration page.
        2. Fill in all required fields with valid data.
        3. Submit the registration form.
    *   **Expected Result:** The system should register the patient successfully and redirect to the login page.

*   **Test Case ID:** TC_002
    *   **Title:** Invalid Patient Registration - Missing Fields
    *   **Description:** Verify that the system handles missing required fields during registration.
    *   **Steps:**
        1. Navigate to the registration page.
        2. Leave one or more required fields empty.
        3. Submit the registration form.
    *   **Expected Result:** The system should display error messages for the missing fields and not register the patient.

*   **Test Case ID:** TC_003

        3. Click the login button.
    *   **Expected Result:** The system should log in the user successfully and redirect to the dashboard.
