**Test Cases:**

*   **Test Case ID:** TC_001
    *   **Description:** Verify that a user can successfully create an account with valid inputs.
    *   **Steps:**
        1. Navigate to the registration page.
        2. Enter valid details for Name, Contact Number, and Email Address.
        3. Click on the "Create Account" button.
    *   **Expected Result:** The user account is created successfully, and the user is redirected to the login page or dashboard.

*   **Test Case ID:** TC_002
    *   **Description:** Verify that a user cannot create an account with an invalid email format.
    *   **Steps:**
        1. Navigate to the registration page.
        2. Enter valid details for Name and Contact Number.
        3. Enter an invalid email format (e.g., "invalid-email").
        4. Click on the "Create Account" button.
    *   **Expected Result:** An error message is displayed, and the account is not created.
